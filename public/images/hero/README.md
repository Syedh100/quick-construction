# Hero Background Image

Place your hero background image here.

## Recommended File
- **Filename**: `hero-background.jpg` or `hero-background.png`
- **Recommended size**: 1920x1080 pixels or larger
- **Format**: JPG (for photos) or PNG (for graphics)

## Usage
The image should be a high-quality photo of a construction/woodworking scene that will be used as the background for the hero section on the homepage.

## Current Image
If you have the woodworking workshop image, save it as:
- `hero-background.jpg` (or `.png`)

The Hero component can be updated to use this image as the background.
