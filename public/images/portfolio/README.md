# Portfolio Images

This directory contains portfolio images for the Quick Fix Construction website.

## Adding Your Images

1. Replace the placeholder images (placeholder-1.jpg through placeholder-9.jpg) with your actual project photos
2. Recommended image size: 1200x900 pixels (4:3 aspect ratio)
3. File formats: JPG, PNG, or WebP
4. Optimize images for web before uploading (keep file sizes under 500KB when possible)
